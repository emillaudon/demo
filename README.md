# Shop Backend

A Spring Boot backend API for a simple shop application.

This project is built as a **learning project** to explore **Spring Boot**, RESTful API design, and backend architecture.  
The focus is on clean structure, domain-driven business logic, and proper handling of orders, products, and stock.

> ⚠️ This project is a work in progress and is still under active development.

## Tech stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- REST / JSON

## Running the project
```bash
mvn spring-boot:run
